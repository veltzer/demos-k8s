
* this exercise should be done only after you have 'kind' and 'kubectl'
	installed and configured with the right versions.

- create a cluster using 'kind'
- describe the nodes in the cluster using 'kubectl describe nodes'
- run an nginx pod on the cluster.
- describe the pod you just ran using 'kubectl describe pods'
- find out how to expose the ports of you pod to the outside network - and do it.
- access you nginx web server from the host.
	use wget
